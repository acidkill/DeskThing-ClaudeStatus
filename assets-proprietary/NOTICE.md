## What's here

- `clawd/` — Clawd pixel-art sprite data (palette + grid JSON).
  - 13 animations originally ingested by the upstream Clawdmeter project from https://claudepix.vercel.app.
  - 2 derivative animations (`idle_strawberry.json`, `idle_bubbles.json`) generated by this repo's `scripts/generate-idle-sprites.mjs`. They reuse the Clawd body pose and palette, so they inherit the same redistribution restrictions as the upstream sprites.
  - **Copyright holders:** Anthropic, PBC and `@amaanbuilds` on X.
  - **License terms:** none granted to redistributors; the upstream Clawdmeter project explicitly flags this as a gray area in its own LICENSING notes.

## Why this directory exists and is committed in this repo

This is a **private fork** maintained by AI-Flow NOWAK for personal use on a single Spotify Car Thing. The sprites are committed so the build pipeline (`scripts/generate-idle-sprites.mjs`, Vite's `import.meta.glob`, and the `.deskthing` packager) can bundle them into the client JS without an external download step.

This repo's `.gitignore` does **not** block `assets-proprietary/` — that block was removed in v0.3.1. If you fork this repo for public release, re-add `assets-proprietary/` to `.gitignore` before pushing to a public remote, and follow the public-fork instructions in `LICENSING.md`.
